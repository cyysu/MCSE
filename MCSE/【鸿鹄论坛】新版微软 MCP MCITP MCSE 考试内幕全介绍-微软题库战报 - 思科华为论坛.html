微软总共分为5个大方向，每个方向下有对应的认证级别


方向一：服务器WindowsServer、ExchangeServer、Lync、SharePoint、MicrosoftAzure

MCSA: Windows Server 2012 认证需要考以下3门考试：
1、Installing and Configuring Windows Server 2012  科目代号410
2、Administering Windows Server 2012 科目代号 411
3、Configuring Advanced Windows Server 2012 Services 科目代号 412

MCSA: Windows Server 2008 认证需要考以下3门考试：
1、Windows Server 2008 Active Directory, Configuring 科目代号 640
2、Windows Server 2008 Network Infrastructure, Configuring 科目代号 642
3、Windows Server 2008, Server Administrator 科目代号 646
（说明：您可以将 MCSA: Windows Server 2008 认证升级到 MCSA: Windows Server 2012 认证。需要考一门Upgrading Your Skills to MCSA Windows Server 2012 科目代号 417）

MCSE: Server Infrastructure 认证需要考以下5门考试：
1、Installing and Configuring Windows Server 2012 科目代号 410
2、Administering Windows Server 2012 科目代号 411
3、Configuring Advanced Windows Server 2012 Services 科目代号 412
（说明：成功完成1 至 3 后，您将获得 Microsoft 认证解决方案专员 (MCSA)：Windows Server 2012 认证。）
4、Designing and Implementing a Server Infrastructure 科目代号 413
5、Implementing an Advanced Server Infrastructure 科目代号 414
（说明：如果您目前拥有 MCSE: Server Infrastructure 认证，并且三年重新认证日期将至，您可以通过参加一门 科目代号980 此考试进行重新认）

MCSE: Desktop Infrastructure 认证需要考以下5门考试：
1、Installing and Configuring Windows Server 2012 科目代号 410
2、Administering Windows Server 2012 科目代号 411
3、Configuring Advanced Windows Server 2012 Services 科目代号 412
（说明：成功完成1 至 3 后，您将获得 Microsoft 认证解决方案专员 (MCSA)：Windows Server 2012 认证）
4、Implementing a Desktop Infrastructure 科目代号 415
5、Implementing Desktop Application Environments 科目代号 416
（如果您目前拥有 MCSE: Desktop Infrastructure 认证，并且三年重新认证日期将至，您可以通过参加一门 科目代号982 此考试进行重新认证）

MCSE: Private Cloud 认证需要考以下5门考试：
1、Installing and Configuring Windows Server 2012 科目代号 410
2、Administering Windows Server 2012 科目代号 411
3、Configuring Advanced Windows Server 2012 Services 科目代号 412
（说明：成功完成1 至 3 后，您将获得 Microsoft 认证解决方案专员 (MCSA)：Windows Server 2012 认证。）
4、Monitoring and Operating a Private Cloud with System Center 2012 科目代号 246
5、Configuring and Deploying a Private Cloud with System Center 2012 科目代号 247
（说明：如果您目前拥有 MCSE: Private Cloud 认证，并且三年重新认证日期将至，您可以通过参加一门 科目代号981 此考试进行重新认证）

MCSE: Messaging 认证需要考以下4门考试：
1、Managing Office 365 Identities and Requirements 科目代号 346
2、Enabling Office 365 Services 科目代号 347
（说明：成功完成 1 和 2 后，您将获得 Microsoft 认证解决方案专员 (MCSA)：Office 365 认证）
3、Core Solutions of Microsoft Exchange Server 2013 科目代号 341
4、Advanced Solutions of Microsoft Exchange Server 2013 科目代号 342

MCSE: Communication 认证需要考以下4门考试：
1、Managing Office 365 Identities and Requirements 科目代号 346
2、Enabling Office 365 Services 科目代号 347
（说明：成功完成 1 和 2 后，您将获得 Microsoft 认证解决方案专员 (MCSA)：Office 365 认证）
3、Core Solutions of Microsoft Lync Server 2013 科目代号 336
4、Enterprise Voice & Online Services with Microsoft Lync Server 2013 科目代号 337

MCSE: SharePoint 认证需要考以下4门考试：
1、Managing Office 365 Identities and Requirements 科目代号 346
2、Enabling Office 365 Services 科目代号 347
（说明：成功完成 1 和 2 后，您将获得 Microsoft 认证解决方案专员 (MCSA)：Office 365 认证）
3、Core Solutions of Microsoft SharePoint Server 2013 科目代号 331
4、Advanced Solutions of Microsoft SharePoint Server 2013 科目代号332
